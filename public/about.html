🛡️ NSFW Shield

AI-Powered Adult Content Detector

Project Overview

NSFW Shield is a modern, privacy-first web application that detects adult (NSFW) content in images directly in the browser using TensorFlow.js.

It analyzes uploaded images and classifies them into five categories: Neutral, Drawings, Hentai, Porn, and Sexy, providing instant visual feedback with clear safe/adult indicators.

How I Built This Project

I created NSFW Shield from scratch with the goal of building a fast, beautiful, and fully client-side AI tool.

Key Highlights:
  • Entirely runs in the browser using TensorFlow.js (No server-side processing)
  • Modern glassmorphism UI with smooth animations
  • Drag & Drop + Click to upload support
  • Real-time image preview and loading states
  • Clear visual distinction between Safe and Adult content

Technical Implementation

• Used TensorFlow.js to load and run the NSFW MobileNet Graph Model
• Preprocessed images (resized to 224×224 and normalized)
• Built responsive UI with pure HTML, CSS, and vanilla JavaScript
• Created a lightweight Express server for easy deployment
